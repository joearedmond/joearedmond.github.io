Hi! My name is Joe.

At work, I do "full stack" machine learning, from exploratory data analysis to modeling to system integration to drift monitoring. The purpose: to help my hospital run smoothly and improve the standard of care for vulnerable patients.

At Columbia, I studied both machine learning and systems programming. In my experience, modeling data alone can prove opportunity, but building a fully-integrated predictive application is where the impact happens. A model in a Jupyter Notebook isn't going to serve itself!
